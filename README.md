# 作者的说明

## 碎碎念

### 开端

2022 年下半年，我使用 Scratch 做了一个游戏，叫“方块迷宫”. 我当时给几个网友分享了这个游戏，其中一个说刚认识我的时候我就在做这个.

音效素材取自[这里](https://github.com/ckcz123/mota-js/tree/master/project/sounds).

然后我一直把游戏的更新计划搁置（（（（（

### 然后……

我在 2023 年接触了 [HTML5 魔塔样板](https://github.com/ckcz123/mota-js/)（https://github.com/ckcz123/mota-js/），然后把它作为一个 RPG 游戏引擎（不仅仅限于魔塔）来用，做出了我在 HTML5 魔塔游戏平台上的第一部作品，得到了很多的反馈.

现在我已经退出 HTML5 魔塔游戏平台了，而那里的方块迷宫也永久停更了.

如果谁还想玩，看这里：[链接](https://h5mota.com/tower/?name=SquareMaze)

*（轻点喷，那都是我以前做的了）*

### 再然后……

我打算用我现有的知识和技术去重写整个游戏，于是这个仓库就诞生了.

我甚至连图片素材都是从原来的作品文件扒的（（（（（
